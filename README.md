## 📢 如何提交反馈
1. 点击上方 **提交反馈** 按钮。  
2. 选择对应的模板：
   - 🐛 **BUG 反馈**
   - 💡 **功能建议**
3. 按要求填写详细信息（截图、坐标、报错信息等越详细越好）。
4. 提交，我们会尽快处理。

---

## 📜 仓库内容
- **更新日志** – 位于 `updates/` 
- **BUG 跟踪** – 所有已知问题在 [Issues](../../issues) 中可见  
- **功能建议** – 玩家可在此提出新想法并讨论  

---

## 📝 提交规则
- 保持礼貌与建设性  
- 使用清晰、简短的标题  
- 一个 Issue 仅限一个问题
